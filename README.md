Каталог товаров магазина

Учебный проект для практики. Разработан на C# с использованием ASP.NET Core Web API, JSON-хранилища и веб-интерфейса.

 Возможности
- Просмотр списка товаров
- Добавление нового товара (название, цена, категория, описание, количество)
- Изменение количества на складе (автоматическое обновление статуса)
- Удаление товара из каталога
- Сохранение данных в JSON-файл

 Стек
- C# .NET 8
- ASP.NET Core Web API
- System.Text.Json
- HTML / JavaScript (Fetch API)

 Запуск
1. Запустите проект `CatalogApi` (через Visual Studio или `dotnet run`).
2. Откройте файл `CatalogUi/index.html` в браузере.
3. Для тестирования API перейдите по адресу `https://localhost:5001/swagger`.
cd CatalogApi //записывать в командной строке
dotnet run    //записывать в командной строке


 Структура
- `Controllers/` – REST API контроллеры
- `Models/` – сущность товара и перечисление статуса
- `Repositories/` – работа с JSON-файлом
- `Services/` – бизнес-логика
- `Data/` – файл `products.json`
- `CatalogUi/` – веб-интерфейс
